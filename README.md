## Project
This project is built with Express js, ejs and Mongodb
## Clone
To clone this project use this command

```bash
git clone https://github.com/MahmoudAhmadOsman/expressjs-RESTful-api.git
```

## Then Install All Dependencies

```bash
npm install --save
```
